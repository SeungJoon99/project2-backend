디렉토리 구조
  폴더 나누는 방법 - 
    각 폴더에 라우터, 서비스, DAO 넣을 예정
    필요하면 ML모델 넣을 예정

지도폴더
  좌표를 받아서 지도를 화면에 출력하는 기능 
  ML X

날씨폴더
  좌표를 받아서 날씨를 화면에 출력하는 기능
  ML X

추천폴더
 날씨를 모델에 넣고 나온 카테고리 / 음식점 반환
  ML O 

검색폴더
  검색창에 검색한 단어 DB에서 조회 후 반환
  ML X

현재 실행 과정
1. 자바스크립트의 fetch가 app에게 url 요청 
2. 해당 url의 라우터가 서비스에게 요청
3. 서비스가 비즈니스 로직 실행 후 
3 - 1. 
  지도 출력이나 날씨 출력의 경우
  해당 API와 송수신 후 결과 반환
3 - 2.
  카테고리 또는 음식점이 필요한 경우 
  DAO에게 요청
  DAO가 DB에서 쿼리 실행 후 결과 반환
